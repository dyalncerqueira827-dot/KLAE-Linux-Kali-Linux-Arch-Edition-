* 💀 Pentest avançado
* ⚙️ Personalização extrema

---

# Instalação

## 1️⃣ Clone o repositório
```bash
git clone https://github.com/dyalncerqueira827-dot/KLAE-Linux-Kali-Linux-Arch-Edition-.git
```

## 2️⃣ Entre na pasta
```bash
cd KLAE-Linux-Kali-Linux-Arch-Edition-
```

## 3️⃣ Dê permissão para os scripts
```bash
chmod +x *.sh
```

## 4️⃣ Execute o instalador
```bash
sudo ./install.sh
```

---

# Requisitos

* Arch Linux instalado
* Conexão com internet
* Usuário com sudo
