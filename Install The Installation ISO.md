# Como instalar o KLAE Linux

O KLAE Linux é uma metadistribuição baseada em Arch Linux.

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/dyalncerqueira827-dot/KLAE-Linux-Kali-Linux-Arch-Edition-.git
```

## 2️⃣ Entre na pasta do projeto

```bash
cd KLAE-Linux
```

## 3️⃣ Dê permissão de execução para o script

```bash
chmod +x klae-install.sh
```

## 4️⃣ Execute o instalador

```bash
sudo ./klae-install.sh
```

O instalador irá automaticamente:

* instalar dependências
* configurar o sistema
* instalar ferramentas de cibersegurança
* configurar o desktop do KLAE Linux
